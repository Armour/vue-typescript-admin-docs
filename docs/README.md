## Getting Started

```bash
# clone the project
git clone https://github.com/armour/vue-typescript-admin-template.git

# enter the project directory
cd vue-typescript-admin-template

# install dependency
yarn # Or npm install

# develop
yarn serve # OR npm run serve
```
